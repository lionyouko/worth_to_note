# Worth To Note

Worth to Note is an example app for the tutorial described in:

## Features in the app

This app is a note taker build using CLEAN architecture to teach people the following concepts:

- separations of concern in each of 4 layers (presentation, application, domain, data);
- use of abstract classes from where entities and models derives;
- freezed (immutable) classes to avoid change mistakes on objects during runtime;
- functional programming with dartz and Either class;

## Libraries used

- flutter_bloc for state management
- freezed for immutability
- dartz for functional programming
- json_serializable for json serialization
- isar for local non-SQL database with reactive capabilities

## Running the App

To run the app, you need to have either Android Studio or VSCode With Flutter extension + the requirements missing when you run flutter doctor.
